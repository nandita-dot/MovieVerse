🎬 MovieVerse – Movie Database Web Application

📌 Overview
    MovieVerse is a movie database web application that allows users to search and explore information about movies in a simple and interactive interface. The project is built using HTML, CSS, and Python, combining frontend design with backend logic to retrieve and display movie data.
The goal of this project is to demonstrate how a full-stack web application can be developed using Python for backend processing and standard web technologies for the user interface.

🚀 Features
  🔎 Search movies by title
  🎥 View movie details such as description, rating, and release information
  📱 Simple and responsive user interface
  ⚡ Fast data retrieval using Python backend
  🎨 Clean UI built using HTML and CSS
